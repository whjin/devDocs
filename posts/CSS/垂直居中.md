# 垂直居中 #

**1、`ling-height`**

适用场景：单行文字垂直居中，按钮、下拉框、导航栏

将单行文字的行高设定后，文字会位于行高的垂直中间位置。

    <div class="content">Long time no see.</div>
    
    .content {
      width: 400px;
      background: #ccc;
      line-height: 100px;
      margin: auto;
    }

**2、`line-height`+`inline-block`**

适用场景：多对象的垂直居中
